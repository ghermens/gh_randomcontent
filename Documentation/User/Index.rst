User's Manual
=============

Setting up a random content element and its content repository
--------------------------------------------------------------

1. Insert a content element of type “General Plugin” into your page.
2. Choose “Random Content” as plugin type.
3. Choose one or more pages as a source for the random content elements.
4. Enter the number of content elements to be shown.
5. Check the “Honor language” box to only use content elements of the current FE language.
6. Check the “Honor Column” box to only use content elements of the same content column as the plugin itself.
